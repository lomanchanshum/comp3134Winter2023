1. What is the definition of server port?
Web server ports are the logical endpoints of a network connection that is used to exchange information 
between a web server and a web client. These ports are assigned a unique number to identify them separately. 
Port 80 and 443 are the two standard ports used by web servers to communicate with the web clients.

2. What is server port range?
Port numbers range from 0 to 65536, but only ports numbers 0 to 1024 
are reserved for privileged services and designated as well-known ports. 
This list of well-known port numbers specifies the port used by the server process as its contact port.
